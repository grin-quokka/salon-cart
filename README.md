# salon-cart

### 배포 : [salon-cart.surge.sh](salon-cart.surge.sh)

### 스택
- typescript
- react, react-router-dom
- [material-ui](https://material-ui.com/) : 디자인

### 기능
- 장바구니에 시술 아이템 추가, 수량 변경, 삭제
- 장바구니에 할인 메뉴 추가, 할인 적용할 시술 변경, 삭제
- 할인이 적용된 최종 금액 계산
